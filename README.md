# simple-restapi
Simple REST API menggunakan Express JS

## Yang dibutuhkan:

 - Node.js 8.2.0 or later
 - Mysql 5.7 or later
 

## Cara Instalasi

    npm install

 

## Cara Menjalankan Aplikasi

    npm run server

Buka alamat [localhost:3000](http://localhost:3000)  pada browser untuk melihat hasilnya.
